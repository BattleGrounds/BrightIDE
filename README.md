# BrightIDE
BrightIDE: simple IDE created(in work) for c++; c; chaiscript; 

# IDE Contol

>**Save**: Ctrl+S

>**Load**: Ctrl+O

>**New**: Ctrl+N

>**Close**: Ctrl+Alt+X

>**Undo**: Ctrl+Y

>**Redo**: Ctrl+Z

>**Select all**: Ctrl+A
